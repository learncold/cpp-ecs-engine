# Tool Icon Candidates

This folder contains the SVG candidates selected for SafeCrowd toolbar icons.
The current application uses the files listed below through `src/application/ToolIcons.qrc`.

## Sources

- Lucide: https://lucide.dev/ and https://github.com/lucide-icons/lucide
- Tabler Icons: https://tabler.io/icons and https://github.com/tabler/tabler-icons

License copies are saved under `licenses/`.
