# SafeCrowd GitHub Project

## 개요
- Project: `SafeCrowd Implementation`
- URL: `https://github.com/users/learncold/projects/1`
- Repository: `learncold/cpp-ecs-engine`
- 관리 방식:
  - GitHub Project와 issues는 필수 개발 절차가 아니라 선택적인 추적 도구로만 사용한다.
  - 작업 시작 전 issue 확인, 생성, 연결은 요구하지 않는다.
  - 구현 단위는 변경 범위, 브랜치, PR, 검증 결과를 기준으로 관리한다.

- Project 항목이나 issue가 없다는 이유로 구현을 지연하지 않는다.
- PR에는 변경 요약, 영역, 아키텍처 점검, 검증 결과를 남긴다.
