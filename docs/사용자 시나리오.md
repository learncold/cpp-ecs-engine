# SafeCrowd 사용자 시나리오

---

## 시나리오 1. 재난 대응 담당자의 비상 상황 대응 훈련 대체 검토

### 사용자
재난 대응 담당자

### 상황
실제 대피 훈련은 비용과 시간이 많이 들고 반복 수행이 어렵다. 담당자는 여러 재난 조건을 가정한 대응 계획을 사전에 검토하고 싶다.

### 목표
- 실제 훈련 전에 다양한 비상 상황을 가상으로 검토한다.
- 사고 가능성이 높은 조건을 식별한다.
- 대응 우선순위와 통제 전략을 수립한다.

### 시나리오 흐름
1. 사용자는 대상 시설의 구조를 시스템에 입력한다.
2. 사용자는 발생할 사고 시나리오와 발생가능 변수를 UI를 통해 편집하여 시스템에 입력한다.
3. 시스템은 입력받은 시나리오를 이용해 시뮬레이션을 생성한다.
4. 사용자가 명령하면 시스템은 시뮬레이션을 실행한다.
5. 시스템은 설정된 수의 시뮬레이션을 병렬로 수행하면 한 눈에 볼 수 있도록 화면에 렌더링 한다.
6. 시스템은 각 시뮬레이션을 진행하면서 낙상, 압사, 탈출 지연, 탈출 실패 같은 위험의 발생 전조가 보이거나 실제 발생했다고 파악한 경우 사용자에게 알린다.
7. 모든 시뮬레이션이 종료되면 시스템은 결과를 요약 및 보고한다.
8. 사용자는 원하는 시뮬레이션 케이스를 선택해 안내 인력 추가, 대체 출구 유도, 통제 구역 재설정 등의 대응안을 반영한다.
9. 수정된 시나리오를 다시 실행해 결과를 비교한다.
10. 가장 효과적인 대응 절차를 비상 대응 매뉴얼 초안에 반영한다.

### 기대 결과
- 반복 훈련 없이도 다양한 재난 조건을 검토할 수 있다.
- 위험 구간과 취약 상황을 빠르게 식별할 수 있다.
- 실제 대응 계획의 완성도를 높일 수 있다.
