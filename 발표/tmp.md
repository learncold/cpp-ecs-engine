
# 편집 툴
## 그리기 툴 자석 기능 추가
모든 그리기 툴은 기존 요소들의 모서리, 변과 잇기 쉽도록 자석 기능을 제공. (재사용 가능한 코드로 작성할 것)

## 다각형 그리기 기능 추가
Room툴은 property panel에서 사각형, 다각형 그리기 모드를 선택할 수 있음.

# Layout 탭
## 레벨 구조 개편
트리 구조를 각 Floor가 최상위에 있도록 그 아래에 Room들이 존재하도록
Room은 그 아래에서 Wall과 Door를 소유함 (공유되는 Wall, Door들도 그냥 각 Room에서 중복해서 표시)
요소들을 클릭하면 렌더링 화면에서 알맞는 floor로 이동해서 하이라이팅. (floor 클릭은 그냥 floor 전환)

# 계단 모델링
## 진입 방향 설정 추가
계단의 진입방향을 설정할 수 있음. (위층과 아래층에서 각각) 나머지 방향으로는 진입이 불가능함 (벽으로 막혀있는 것과 같음)

## 렌더링 수정
진입방향 화살표로 표시, 위층과 이어지면 Up, 아래층과 이어지면 Down이라고 텍스트 렌더링

## 시뮬레이션 상에서
이동 속도가 느려지는 통로처럼 여김, 절반을 넘어가면 다음 층에서 보행자가 렌더링 되도록 변경
