# 졸업 프로젝트 주차별 계획서

## 1. 프로젝트 개요
- **주제**: C++ DOD/ECS 기반 데이터 지향 설계를 적용한 고성능 게임 엔진 구현 및 대규모 군집 지능 시뮬레이션 연구
- **기간**: 2026년 3월 ~ 11월 (2학기)
- **핵심 목표**
  1. C++ 기반 ECS 엔진 코어 설계 및 구현
  2. 대규모 Boids(군집 지능) 시뮬레이션 구현
  3. OOP 방식과 ECS 방식의 성능 비교 및 최적화

## 2. 추진 방향
업로드된 계획서의 월별 흐름은 다음과 같다.
- **1~2월/초기 단계**: DOD, ECS(Archetype/Sparse Set) 이론 학습 및 메모리 구조 설계
- **3~6월**: Entity/Component 중심의 엔진 코어 및 대량 렌더링 구현
- **7~8월**: OOP 대조군 개발 및 Boids 시뮬레이션 적용
- **9~11월**: Job System, 성능 지표(FPS, CPU Time, 캐시 효율) 기반 최적화·벤치마킹·발표 준비

본 문서는 위 흐름을 바탕으로, **아직 프로젝트를 시작하지 않은 상태**를 기준으로 바로 실행 가능한 주차별 계획으로 재구성한 문서이다.
